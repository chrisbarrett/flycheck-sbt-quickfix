# flycheck-sbt-quickfix

## Summary

Flycheck checker for sbt-quickfix plugin.

## Installing

You will need Emacs 24+, `make` and [Cask](https://github.com/cask/cask) to
build the project.

    cd flycheck-sbt-quickfix
    make && make install
